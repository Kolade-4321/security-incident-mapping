# Security Incident Mapping (SIM)

## Overview

Security Incident Mapping (SIM) is a simple web application developed using HTML, CSS, and JavaScript. The system allows users to anonymously report security incidents and displays submitted reports in a live incident feed without reloading the page.

## Features

* Anonymous incident reporting
* Crime type selection
* Incident location input
* Incident description input
* Live incident feed updates
* Responsive and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run the Project

1. Download or clone the project files.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Fill in the report form and click **Submit Report**.
5. The submitted report will immediately appear in the Live Incident Feed.
